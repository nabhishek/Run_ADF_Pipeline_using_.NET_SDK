# Run_ADF_Pipeline_using_.NET_SDK
Simple C# code to trigger an existing ADF V2 pipeline.

Please update the following in the code (Program.cs) to be able to successfully run your ADFv2 pipeline.
  1. tenantID
  2. applicationId 
  3. authenticationKey
  4. subscriptionId
  5. resourceGroup
  6. dataFactoryName
  7. pipelineName
